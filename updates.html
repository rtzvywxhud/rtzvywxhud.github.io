    Split Text tool added to index page.

    Added an option to set default font on every text tool.

    Added options to enable/disable spell checker and word wrap.

    Added Password and Filler Text generator tool.

  • New
    Added Commafy Numbers tool.
April 22, 2018
  • New
    Dark/night mode added in user settings page.
April 21, 2018
  • Improved
    Applied modern Durstenfeld algorithm when randomizing/shuffling a list in Sort Text tool.
  • Improved
    Added ascending and descending order in Sort Text tool. Now you can sort a list in descending order directly instead of sorting alphabetically and then using the reverse function.
  • Improved
    Added title case and AP title case in Convert Case tool.
April 6, 2018
  • Fixed
    Fixed an issue in Concatenate Text tool, where it would stop merging the lists when column B contains more lines than column A, and vice versa.
April 4, 2018
  • Improved
    Added "whole words only" and "match case" options in Find and Replace tool.
April 2, 2018
  • New
    Added "Copy to Clipboard" function on every tool. You can now copy the content of the textarea easily by clicking the icon on the top right corner.
March 30, 2018
  • New
    Added undo and redo function which can be found under edit button.
March 29, 2018
  • Improved
    Added print function under file menu. You can now directly print the content of the working area.
  • New
    Added file upload/download function. You can now directly open and save plain text documents (UTF-8) from your computer.
March 16, 2018
  • Improved
    Improved site layout using our own framework and added material icons for better user interface.

  • New
